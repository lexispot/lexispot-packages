# LexiSpot Language Packages

This repository hosts dictionary packages for the LexiSpot iOS app.

## Available Languages

34+ language packages for offline translation.

## Download

All packages are available in the [Releases](https://github.com/lexispot/lexispot-packages/releases) section.

## Format

- **Format**: SQLite database
- **Compression**: Zstandard (.zst)
- **Verification**: SHA256 checksums

## Usage

These packages are automatically downloaded by the LexiSpot iOS app. Users can select and download only the languages they need.
